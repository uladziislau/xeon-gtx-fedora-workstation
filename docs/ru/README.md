# 📚 Индекс Документации

Полная документация по оптимизации Fedora Workstation на системах Intel Xeon E5 v3 + NVIDIA GTX.

## 📖 Содержание

### 🛠️ Системные настройки
- **[Настройка ядра](system/core-setup.md)** - Параметры ядра, конфигурация GRUB и управление питанием
- **[Планировщики CPU](system/schedulers.md)** - Тестирование и сравнение планировщиков sched-ext для 36-поточных систем

### 🎨 Графика и дисплей
- **[Графика и дисплей](graphics/display.md)** - Конфигурация NVIDIA + Wayland + HDR

### 🌐 Браузеры
- **[Zen Browser](browsers/zen.md)** - Руководство по настройке Zen Browser (форк Firefox)
- **[Thorium Browser](browsers/thorium.md)** - Конфигурация Chromium с оптимизацией AVX2

### 🆘 Решение проблем
- **[Журнал проблем](troubleshooting/index.md)** - Частые проблемы и их решения

### 💻 Справочник
- **[Справочник команд](cli-cheatsheet.md)** - Быстрая справка по командам для управления системой

## 🚀 Быстрый старт

1. **Системные настройки**: Начните с [Настройки ядра](system/core-setup.md) для конфигурации параметров ядра
2. **Графика**: Настройте [Графика и дисплей](graphics/display.md) для поддержки NVIDIA Wayland
3. **Браузеры**: Примените конфигурации браузеров из раздела [Браузеры](browsers/)
4. **Решение проблем**: Ознакомьтесь с [Журналом проблем](troubleshooting/index.md) при возникновении вопросов
