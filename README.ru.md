# Конфигурация Zen Browser для рабочей станции

[English version](README.md)

Высокопроизводительный `user.js` для Zen Browser под Linux.  
Оптимизирован для максимальной отзывчивости, стабильных 165 Гц и тяжёлых SPA/WebGPU‑нагрузок.

## Аппаратная платформа
- 2560×1440 @ **165 Гц**
- Intel Xeon (18 ядер / 36 потоков)
- **64 ГБ RAM**
- NVIDIA **GTX 1660 Super**
- NVMe SSD
- Fedora Linux (Wayland, KDE Plasma)

## Основные особенности
- Полный WebRender (Warp, DMA‑BUF, без partial present)
- WebGPU на Vulkan
- Агрессивное кэширование в RAM (4 ГБ + 2 ГБ decoded images)
- 24 процесса контента, 8 процессов расширений
- Оптимизированный IPC‑планировщик (8 потоков, preemption)
- Wayland vsync + opaque region
- Плавный скроллинг под 165 Гц
- Телеметрия отключена

## Установка
Переименуйте `user.ru.js` в `user.js`.
Поместите `user.js` в профиль Zen Browser:
```
~/.zen/<профиль>/user.js
```

Перезапустите браузер.

## Примечания
Конфигурация рассчитана на мощные рабочие станции.  
Не рекомендуется для слабых ПК и ноутбуков.
