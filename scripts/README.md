# 🚀 Automation Scripts

> Скрипты автоматизации для быстрой настройки и оптимизации системы.

## 📋 Скрипты

### `optimize.sh` - Главный скрипт оптимизации

Автоматически применяет все необходимые настройки:

- ✅ Параметры загрузки GRUB (kernel parameters)
- ✅ Системные переменные окружения (`/etc/environment`)
- ✅ Конфигурация Zen Browser (`user.js`)
- ✅ Профиль питания (Performance)

**Использование:**

```bash
# Тестовый запуск (без изменений)
./scripts/optimize.sh --dry-run

# Реальный запуск (требует root)
sudo ./scripts/optimize.sh
```

**Что делает скрипт:**

1. Создает резервные копии всех изменяемых файлов в `~/.xeon-gtx-backups/`
2. Настраивает параметры ядра в GRUB
3. Добавляет переменные окружения для NVIDIA/Wayland/HDR
4. Устанавливает `user.js` для Zen Browser (если профиль найден)
5. Устанавливает профиль производительности

**После выполнения:**

```bash
# Обновить GRUB
sudo grub2-mkconfig -o /boot/grub2/grub.cfg

# Перезагрузить систему
sudo reboot
```

### `check.sh` - Проверка текущих настроек

Проверяет, какие настройки уже применены:

```bash
./scripts/check.sh
```

## ⚠️ Требования

- Fedora 43+ (Rawhide)
- KDE Plasma 6
- NVIDIA Driver 580+
- Root доступ (для системных настроек)

## 🔒 Безопасность

- Все изменяемые файлы автоматически бэкапятся
- Скрипты используют `set -euo pipefail` для безопасного выполнения
- Проверка прав root перед системными изменениями
